    - Общий счёт и кнопки "Выйти", "Повторить"/"Продолжить"

Игрок выбирает клетку.
Перед выстрелом - вопрос из пула easy? по выбранным темам.
Ответ неверный - ход переходит к сопернику(для начала просто функция, которая рандомно бьет по доступным клеткам игрока).
Ответ верный - проверка поля на наличие корабля.
Клетка пустая - какая-нибудь плюшка за правильный ответ + помечается, что вопрос отвечен в банке вопросов, прогресс по изучению темы.
Клетка с кораблем - вопрос по теме корабля.
Правильный ответ - попал, следующий вопрос по этой теме(сложнее? другой тип? оставить просто рандом?) - правильный ответ - попал, пока весь корабль не потоплен.
Новый выстрел в поиске других кораблей.

За каждый правильный ответ - очки по этой теме? Суммарно? Соответственно ранги просто как плюшки при игре с авто противником, если добавлять второго игрока то уже ранг для выбора игрока по уровню?
