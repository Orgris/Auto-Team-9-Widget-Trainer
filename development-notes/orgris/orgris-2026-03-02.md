## Start/About Screen

Цель: Заинтересовать, объяснить правила, конвертировать в регистрацию

- Header (Глобальная навигация)
  - Logo — клик возвращает на начало
  - Nav Links — якоря для плавного скролла:
    - О проекте
    - Как играть
    - Команда
    - FAQ
  - User Widget (Conditional):
    - Гость: Кнопки «Войти» / «Регистрация»
    - Авторизован: Аватар + Ник + Кнопка «Выйти»
  - Mobile Toggle — гамбургер-меню для мобильных

- Hero Section
  - Visual — атмосферный арт (остров/корабль + код)
  - Copy:
    - Заголовок (H1): [УТП](https://tilda.education/unique-selling-proposition) (например, «Покори TypeScript в морском бою»)
    - Подзаголовок: Краткая суть механики («Решай задачи — топи корабли»)
  - CTA Buttons:
    - Primary: «Играть» (Логика: если авторизован → Dashboard, если нет → Auth)
    - Secondary: «Как играть» (скролл к правилам или модальное окно)

- Stats Bar
  - Counters: Анимированные цифры: Задач, Тем, Пользователей, Побед.

- "Why Us" / Features (Преимущества)
  - Cards Grid: Иконка + Заголовок + Текст.
    - Геймификация: Учеба не душит, а азартит.
    - Адаптивность: Играй с любого устройства.
    - RSS Approach: Качественный контент от школы.
    - ...

- How to Play
  - Steps: Простая схема (1. Расставь флот → 2. Выбери клетку → 3. Ответь на вопрос → 4. Победа).
  - Visuals: Скриншоты или схематичные иконки этапов.

- Team Section
  - Members: Аватар, Имя, Роль, Ссылки (GitHub).

- FAQ (Аккордеон)
  - Questions: Технические (какой стек?), игровые (что если ответил неверно?), организационные.
  - Interaction: Раскрытие ответа по клику.

- Footer
  - Info: Копирайт, Ссылка на RSS School, Политика конфиденциальности.
  - Socials: Иконки Discord, GitHub, Telegram.

## Auth Screen

Цель: Быстрый и безопасный вход. Минимизировать трение.

- Layout Container
  - Center Card: Форма по центру, чистый фон или размытый арт.
  - Branding: Логотип и слоган в шапке карточки.

- Auth Methods ([Firebase](https://firebase.google.com/docs/auth/web/start?hl=ru))
  - Email Form (Fallback):
    - Поля: Email, Password.
    - Validation: Проверка формата email, мин. длина пароля (визуальная обратная связь).
    - Кнопка: «Войти».
  - Divider: Визуальный разделитель «Или войти через email».
  - Social Providers (Priority):
    - Кнопки «Continue with [Google»](https://firebase.google.com/docs/auth/web/google-signin?hl=ru) и «Continue with [GitHub»](https://firebase.google.com/docs/auth/web/github-auth?hl=ru).
    - Логика: OAuth flow через Firebase.

- Recovery & Toggle
  - Links: «Забыли пароль?» (открывает модалку сброса) и «Нет аккаунта? Зарегистрироваться» (переключатель режимов формы).

- Feedback & UX
  - Loading States: Блокировка кнопок и спиннер во время запроса к Firebase.
  - Error Handling: Понятные сообщения об ошибках (не «Error 404», а «Неверный пароль» или «Пользователь не найден»).
  - Success Redirect: Плавный переход в Dashboard после успеха.

## Dash Screen / Тактический стол

Цель: Дать игроку обзор прогресса, мотивировать к следующему шагу и предоставить быстрый доступ к игре.

- Блок 1: Player Status Panel (Верхняя панель)
  - Профиль-сводка
    - Mini Profile Card — быстрый доступ к информации:
      - Аватар + Ник + Ранг
      - Функция перехода в профиль
  - Ранг и прогресс
    - Current Rank Badge — иконка + название (Junior, Middle, Senior, Captain и т.д.)
    - Progress Ring — круговая диаграмма:
      - Заполнение: XP текущий / XP до следующего звания
      - Tooltip при наведении: «Осталось 120 XP до Middle»
  - Активность и серии
    - Streak Counter — «🔥 7 дней в море»:
      - Иконка огня/якоря
      - Число текущей серии
      - Бонус: Визуальный индикатор, если сегодня ещё не было активности
    - Last Visit — «Вы были в море: сегодня / вчера / 3 дня назад»

- Блок 2: Archipelago Map (Основная зона)
  - Визуализация прогресса
    - Map Container — стилизованная сетка/карта/радар в морском стиле:
      - Серые клетки — заблокированные уровни (требуется пройти предыдущий)
      - Закрашенные/«подбитые» клетки — пройденные уровни
      - Цветные клетки — доступные для атаки уровни
    - Hover Effects — при наведении на клетку:
      - Название уровня
      - Сложность (⭐⭐⭐)
      - Статус: «Пройдено», «Доступно», «Закрыто»
  - Переключение тем (карт)
    - Topic Tabs / Dropdown — выбор темы = смена карты:
      - TypeScript Basics | Types | Async | OOP | Generics ...
      - Визуальный индикатор: название выбранной сейчас темы
      - Кнопки «⮂» для переключения карты
      - Опционально: Прогресс по каждой теме (% пройденных уровней)
    - Map Navigation — если уровней много:
      - Кнопки «⮃ или ⮂» для скролла карты

- Блок 3: Action Zone (Призыв к действию)
  - Кнопка «ОГОНЬ» (Start Coding)
    - Primary CTA — огромная, яркая, с анимацией:
    - Текст: «🔥 ОГОНЬ» или «⚓ В БОЙ»
    - Hover-эффект: свечение / пульсация / звук (опционально)
    - Click-логика:
      - Если уровень выбран → запуск игры
      - Если нет → подсказка «Выберите цель на карте»

- Блок 4: Daily Quests / Приказы
  - Список заданий на день
    - Quest Card — компактный блок с заданиями
    - Progress Indicators — чекбоксы / прогресс-бары
      - 📋 Ежедневные приказы:
        - ☐ Решить 1 задачу (0/1) [+10 XP]
        - ☐ Решить задачу без ошибок [+15 XP]
        - ☐ Прочитать 1 статью теории [+5 XP]
    - Reward Badge — визуальное отображение награды за выполнение
    - Reset Timer — «Обновление через: 14:32:10»

- Блок 5: Leaderboard / Таблица результатов
  - Топ игроков
    - Table Headers:
      - № | Игрок | Ранг | Уровень | XP
    - Current User Highlight — строка текущего игрока выделена, даже если не в топе
    - Scope Toggle — переключатель:
      - «Глобальный рейтинг» / «Среди друзей» / «На этой неделе»

- Блок 6: Дополнительные элементы (опционально)
  - Быстрые настройки
    - Settings Gear — иконка в углу:
      - Звук вкл/выкл
      - Сложность вопросов (Легко/Нормально/Хардкор)
      - Тема интерфейса (Светлая/Тёмная/Морская)
  - Уведомления и достижения
    - Toast Notifications — всплывающие сообщения:
      - «🎉 Новый ранг получен!»
      - «📦 Ежедневная награда доступна»
      - Achievement Badges — мини-иконки за достижения:
        - «Первая победа», «Серия 7 дней», «Без ошибок»

## Profile Screen / Личное дело

Цель: Отобразить прогресс игрока, дать чувство достижения и возможность кастомизации. Это «лицо» игрока в системе.

- Блок 1: Визитка (Identity Card)
  - Аватар
    - Display: Квадтраное изображение.
    - Upload: Возможность загрузить своё фото.
    - Фишка: Автоматический фильтр «Пиксель-арт» при загрузке.
    - Crop Tool: Простая обрезка квадрата перед загрузкой.
    - Default: Стандартная аватарка (силуэт моряка/робота), если пользователь не загрузил свою.
  - Позывной (Имя)
    - Display: Отображение имени пользователя (из Auth профиля или редактируемое).
    - Edit: Возможность изменить отображаемое имя.
    - Validation: Уникальность ника, запрет спецсимволов.
  - Звание (Ранг)
    - Visual: Красивая плашка с иконкой звания.
    - Hierarchy (Прогрессия):
      - 🐣 Юнга (Start)
      - 🧢 Мичман (Junior)
      - ⭐ Лейтенант (Middle)
      - ⚓ Капитан 3 ранга (Senior)
      - 🦅 Адмирал флота (Expert)
    - Прогресс бар: Сколько XP до следующего звания.
  - Флаг / Герб (Клановая система)
    - Visual: Иконка рядом с аватаром или фоном визитки.
    - При клике выводит на страницу кланов

- Блок 2: Судовой Журнал (Статистика и История)
  - Основная статистика
    - Потоплено целей: Количество решенных задач (Total Solved).
    - Точность стрельбы: Процент решений с первого раза (Accuracy %).
    - Время в море: Общее время, потраченное на игру.
    - Лучшая серия побед подряд (Best Streak).
  - Достижения / Бейджи
    - Grid View: Сетка иконок достижений.
    - States:
      - 🟢 Разблокировано: Цветная иконка + подпись.
      - ⚫ Закрыто: Серая иконка + условие («Решить 5 задач подряд»).
    - Examples:
      - «Первая кровь» (1 победа)
      - «Снайпер» (10 задач без ошибок)
      - «Морской волк» (30 дней активности)

- Блок 3: Кастомизация (Внешний вид)
  - Скины приложения (Темы)
    - Options:
      - «Классический» (Светлый/Тёмный)
      - «Морская ночь» (Синяя тема)
      - «Ретро-терминал» (Зелёный на чёрном)
      - Unlock Condition: Некоторые темы открываются за достижения.
  - Стиль кораблей
    - Visual: Иконки, которые отображаются в бою.
    - Models:
      - «Стандарт»
      - «Пират» (За 100 побед)
      - «Пиксель» (За звание Адмирал)
  - Эффекты победы
    - Animations:
      - «Салют» (Конфетти)
      - «Взрыв» (Пиксельная анимация)
      - «Тишина» (Без эффектов)

## Clan Screen:

Цель: Создать ощущение принадлежности к команде и здоровой конкуренции без сложной механики кланов. Игрок выбирает одну из заранее созданных фракций и видит общий вклад своей стороны.
Концепт: Фиксированные фракции. Игрок не создаёт клан, а присоединяется к одной из сторон.

- Блок 1: Выбор стороны (Faction Selector)
  - Карточки фракций - Visual: Крупные карточки с гербом, цветовой схемой и описанием.
    - 🛡️ ГЕРБ (SVG/PNG)
    - Название: «Алый Прилив»
    - Motto: «Атакуй, пока другие читают документацию»
    - [ Присоединиться ]
  - Description: Краткий лор + философия фракции (1-2 предложения).
  - Preview Stats: Мини-превью статистики прямо на карточке (игроки, XP).
- Механика выбора
  - One-time Choice: Игрок выбирает фракцию один раз при первом входе.
  - Cooldown / Cost: (опционально) Смена фракции возможна раз в неделю / за внутриигровую валюту.
  - Confirmation Modal: «Вы уверены? Ваш прогресс сохранится, но вклад в статистику начнётся с нуля».

- Блок 2: Глобальная статистика (War Dashboard)
  - Сводная таблица сторон
    - Headers: Фракция | Игроков | Суммарный XP | Часов в бою | Побед
    - Rows: Динамические данные по каждой фракции
    - Visual Feedback:
      - Прогресс-бар «Кто лидирует» (по XP или победам)
      - Анимация «пульса/огня» у лидирующей фракции
  - Динамика и тренды
    - Time Filter: «За сегодня» / «За неделю» / «Всё время»
    - Chart (опционально): Линейный график активности фракций.
    - Momentum Badge: «🔥 Красные набрали 5000 XP за последние 24ч»

- Блок 3: Личный вклад игрока (My Contribution)
  - Персональная статистика в контексте фракции
    - Display: «Ваш вклад в победу [Название фракции]»:
      - XP, добавленный в общий зачёт
      - Задач решено за фракцию
      - Место в рейтинге фракции («Вы в топ-15% своей стороны»)
    - Rank Badge: Мини-звание внутри фракции (Рекрут → Ветеран → Элита).
  - Призыв к действию
    - CTA Text: «Ваша фракция отстаёт на 12 000 XP! Решите задачу, чтобы помочь».
    - Quick Button: «В бой» → переход на Dash Screen с предвыбранной темой.

- Блок 4: Рейтинг игроков внутри фракции (опционально)
  - Топ-10 contributors
    - Table: # | Игрок | XP | Задач | Серия
    - Scope: Показывает топ только внутри выбранной фракции.
  - Награды фракци (опционально)
    - Weekly Winner: Фракция-победитель сезона получает:
      - Уникальный бейдж в профиль
      - Временный скин / эффект
      - Право «называть следующую тему недели» (геймификация)

## Game Screen:

- Игровое поле (Board System)
  - Два поля 10×10
    - Player Board — поле игрока (расположение своих кораблей).
    - Enemy Board — поле противника (скрыто, открывается в процессе атаки).
    - Grid Size: 10×10 клеток (стандарт морского боя).
    - Cell States:
      - empty — пустая клетка
      - ship-part — корабль
      - miss — промах (флаг/точка)
      - hit — попадание (крестик/огонь)
      - sunk — потоплен (зачёркнутый корабль)
      - unknown — туман войны (на поле противника)
  - Переключение между полями
    - Swap Button — кнопка переключения вида между полями с анимацией (flip/flip).
    - Mini-map — миникарта своего поля в правом верхнем углу (постоянно видна).

- ⚓ Game Screen — Prep Stage (Этап подготовки)
  - Расстановка кораблей
    - Drag & Drop — перетаскивание кораблей из «дока» на поле.
      - Корабли: 1×4, 2×3, 3×2, 4×1 (классический набор).
      - Валидация: нельзя ставить внахлёст, нельзя ближе 1 клетки к другому кораблю.
    - Rotation — поворот корабля перед установкой (кнопка R или иконка).
    - Dock — панель с доступными кораблями (показывает, что ещё не размещено).
    - Controls:
      - «Авто-расстановка» (рандом, но валидный) — опционально
      - «Очистить поле» — сбросить все корабли
      - «В бой!» — завершить подготовку (активна, когда все корабли размещены)
  - Таймер подготовки (опционально)
    - Countdown: 60 секунд на расстановку.
    - Penalty: Если время вышло — авто-расстановка.

- ⚔️ Game Screen — Game Stage (Этап боя) «выстрел = вопрос».
  - Ход игрока
    - Cell Selection — клик по клетке на поле противника для выбора цели.
      - Подсветка выбранной клетки.
      - Блокировка уже «открытых» клеток (нельзя стрелять дважды в одно место).
    - Fire Button — кнопка «Выстрел» или клавиша Enter для подтверждения.
      - Альтернатива: Выстрел сразу по клику (быстрее, но больше случайных нажатий).
    - Turn Indicator — визуальный маркер «Ваш ход» / «Ход противника».
  - Модалка с вопросом (Quiz Modal)
    - Trigger: Открывается при выстреле по кораблю противника.
    - Content:
      - Вопрос по выбранной теме (из пула уровня/сложности).
      - Варианты ответа (Multiple Choice) или ввод кода.
      - Таймер на ответ (опционально, 30-60 сек).
    - Results:
      - ✅ Правильно: Попадание по клетке → игрок ходит снова.
      - ❌ Неправильно: Промах/корабль цел → ход переходит противнику.
    - Feedback: Анимация правильного/неправильного ответа.
  - Расходники / Способности (Power-ups)
    - Inventory Bar — панель с доступными расходниками.
    - Examples:
      - 🎯 Прицел: «Просвет» зоны 3×3 на поле противника (показать, есть ли корабли).
      - 🛡️ Щит: Защита одного своего корабля от следующего попадания.
      - 🔄 Сдвиг: Переместить один свой корабль во время боя.
    - Counters: Количество использования (лимит за игру).
    - Unlock: Открываются за достижения или внутриигровую валюту.
  - Ход противника (AI)
    - Logic: Простой алгоритм для начала.
      - Случайный выбор клетки из доступных.
      - Улучшение: Если попал — стреляет рядом (добивает корабль).
    - Timing: Задержка 1-3 секунды перед ходом (имитация «думания»).
    - Feedback: Визуализация выстрела противника на поле игрока.

- 🏆 Game Screen — End Stage (Результаты)
  - Цель: Показать итоги, наградить, мотивировать к повторению.
  - Победа / Поражение
    - Banner: Крупный визуальный баннер с результатом.
      - 🎉 «ПОБЕДА!» + анимация салюта/взрыва.
      - 💀 «ПОРАЖЕНИЕ» + анимация тонущего корабля.
    - Stats Summary:
      - Точность стрельбы (% попаданий).
      - Правильных ответов / Всего вопросов.
      - Время боя.
      - Использовано расходников.
  - Список отвеченных вопросов
    - Review List: Таблица с вопросами из этой игры.
      | Вопрос | Ваш ответ | Правильный | Очки |
      |--------|-----------|------------|------|
      | ... | ✅ Да | ✅ Да | 100 |
      | ... | ❌ Нет | ✅ Да | 0 |
    - Learning Value: Игрок может увидеть правильные ответы на ошибки.
  - Система очков (Scoring)
    - Formula: Чем меньше ошибок — тем больше очков
      - Идеально (0 ошибок) × 7 = 700 XP
      - Хорошо (1-2 ошибки) × 5 = 400 XP
      - Нормально (3+ ошибок) × 3 = 200 XP
    - Bonus:
      - Серия побед: +10% к XP.
      - Быстрая победа: +5% к XP.
      - Без расходников: +15% к XP.
    - Total Display: Крупно показать итоговый XP + прогресс до следующего звания.
  - Действия после игры
    - Buttons:
      - «Выйти» → возврат на Dash Screen.
      - «Повторить» → новая игра с теми же настройками.
      - «Продолжить» → следующий уровень на карте (если есть).
    - Share: (опционально) Кнопка «Поделиться результатом» в соцсети.
